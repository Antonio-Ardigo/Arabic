# 2. زين (zain) — "good / fine / nice"

| Field | Value |
|---|---|
| Arabic | زين |
| Transliteration | zain |
| English | good / fine / nice |
| Forvo | [زين](https://forvo.com/word/زين/#ar) |
| Local audio | [zain.mp4](../audio/zain.mp4) |

Najdi default for "good". MSA equivalent is جيد but Saudis use `zain` in speech.

## Example 1
**Arabic:** أنا زين، الحمد لله
**Transliteration:** ana zain, al-hamdu lillah
**English:** I'm fine, thank God.

| Arabic | Transliteration | English |
|---|---|---|
| أنا | ana | I |
| زين | zain | fine |
| الحمد | al-hamdu | the-praise |
| لله | lillah | to-God |

## Example 2
**Arabic:** الأكل زين مره
**Transliteration:** al-akl zain marra
**English:** The food is really good.

| Arabic | Transliteration | English |
|---|---|---|
| الأكل | al-akl | the-food |
| زين | zain | good |
| مره | marra | very |
