# 5. أبغى (abgha) — "I want"

| Field | Value |
|---|---|
| Arabic | أبغى |
| Transliteration | abgha |
| English | I want (Najdi/Hijazi; MSA is أريد) |
| Forvo | [أبغى](https://forvo.com/word/أبغى/#ar) |
| Local audio | [abgha.mp4](../audio/abgha.mp4) |

Stem: `bgha`. Conjugates: `abgha` (I), `tibgha` (you m.), `tibghiin` (you f.), `yibgha` (he), `tibgha` (she), `nibgha` (we).

## Example 1
**Arabic:** أبغى ماء
**Transliteration:** abgha maay
**English:** I want water.

| Arabic | Transliteration | English |
|---|---|---|
| أبغى | abgha | I-want |
| ماء | maay | water |

## Example 2
**Arabic:** أبغى أروح البيت
**Transliteration:** abgha aruh al-bait
**English:** I want to go home.

| Arabic | Transliteration | English |
|---|---|---|
| أبغى | abgha | I-want |
| أروح | aruh | I-go |
| البيت | al-bait | the-house |
