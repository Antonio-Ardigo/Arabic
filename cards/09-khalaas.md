# 9. خلاص (khalaas) — "done / enough / okay"

| Field | Value |
|---|---|
| Arabic | خلاص |
| Transliteration | khalaas |
| English | done / enough / that's settled |
| Forvo | [خلاص](https://forvo.com/word/خلاص/#ar) |
| Local audio | [khalaas.mp4](../audio/khalaas.mp4) |

Multi-use discourse marker. As a conversation closer ("okay, that's that"), as "enough!" to stop something, or as an adjective for "finished".

## Example 1
**Arabic:** خلاص، فهمت
**Transliteration:** khalaas, fihamt
**English:** Okay, I understood.

| Arabic | Transliteration | English |
|---|---|---|
| خلاص | khalaas | okay / done |
| فهمت | fihamt | I-understood |

## Example 2
**Arabic:** خلاص الشغل؟
**Transliteration:** khalaas ash-shughul?
**English:** Is the work finished?

| Arabic | Transliteration | English |
|---|---|---|
| خلاص | khalaas | finished |
| الشغل | ash-shughul | the-work |
