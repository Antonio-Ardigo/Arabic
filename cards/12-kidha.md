# 12. كذا (kidha) — "like this"

| Field | Value |
|---|---|
| Arabic | كذا |
| Transliteration | kidha |
| English | like this / like that / so |
| Forvo | [كذا](https://forvo.com/word/كذا/#ar) |
| Local audio | [kidha.mp4](../audio/kidha.mp4) |

Najdi variant `chidhi` also exists (with ك → `ch` affrication); `kidha` is the neutral written form.

## Example 1
**Arabic:** سوّها كذا
**Transliteration:** sawwha kidha
**English:** Do it like this.

| Arabic | Transliteration | English |
|---|---|---|
| سوّها | sawwha | do-it (f., imp.) |
| كذا | kidha | like-this |

## Example 2
**Arabic:** ليش كذا؟
**Transliteration:** laish kidha?
**English:** Why like this?

| Arabic | Transliteration | English |
|---|---|---|
| ليش | laish | why |
| كذا | kidha | like-this |
