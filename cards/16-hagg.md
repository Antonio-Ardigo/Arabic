# 16. حق (hagg) — "belonging to / for"

| Field | Value |
|---|---|
| Arabic | حق |
| Transliteration | hagg |
| English | belonging to / for (Najdi possessive) |
| Forvo | [حق](https://forvo.com/word/حق/#ar) |
| Local audio | [hagg.mp4](../audio/hagg.mp4) |

Najdi possessive particle (Gulf `maal` is more common elsewhere). Takes pronoun suffixes: `haggi` (mine), `haggak` (yours m.), `haggha` (hers), etc. Literally "right / due", used structurally for ownership.

## Example 1
**Arabic:** هذا حقي
**Transliteration:** hatha haggi
**English:** This is mine.

| Arabic | Transliteration | English |
|---|---|---|
| هذا | hatha | this (m.) |
| حقي | haggi | mine |

## Example 2
**Arabic:** السيارة حق أخوي
**Transliteration:** as-sayyaara hagg akhooy
**English:** The car belongs to my brother.

| Arabic | Transliteration | English |
|---|---|---|
| السيارة | as-sayyaara | the-car |
| حق | hagg | belonging-to |
| أخوي | akhooy | my-brother |
